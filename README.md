## Contribute Guide

Requirement:

- [node](https://nodejs.org/): `^15.11.0`
- [pnpm](https://pnpm.js.org/): `^5.18.3`

Install dependencies:

```sh
pnpm install
```

Run development server:

```sh
pnpm run dev
```
