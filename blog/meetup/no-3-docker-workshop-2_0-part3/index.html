幹話#3 - Docker workshop 2.0

2021年11月24日 · 1 分鐘閱讀
Miles Chou
幹話王

遲來的 11/18 第三次的 Docker 閒聊，總算跑出 Dockercraft 了。

再繼續前一次的 network，這次講 Docker Compose 的相關應用，以及 demo Dockercraft。就經驗來說，如果想要跑 Dockercraft 的話，就需要花錢買 Minecraft；使用 Lima 跑 Dockercraft 的話，會需要權限存取 host 的 docker.sock 檔，因此需要進入 Lima 並 sudo 成 root 權限才能正常執行。

下一次會由 Gson 來聊聊 NestJS，若有想分享的主題，歡迎來 Discord 或 GitHub Discussions 一起討論哦！
